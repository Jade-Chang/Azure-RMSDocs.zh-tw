# 支援的平台

本主題說明支援 Rights Management Services SDK 2.1 的用戶端和伺服器平台。

## 支援的平台

-   Windows 10
-   Windows Server 2012 R2
-   Windows Server 2012
-   Windows 8
-   Windows Server 2008 R2 SP1
-   Windows 7 + SP1

**注意**：Windows Server 2008 開始支援使用範本功能。
